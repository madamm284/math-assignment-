# Mean Calculation in JavaScript

This small JavaScript exercise calculates the average (mean) of the numbers stored in an array.

## Concepts practiced
- Arrays
- For loops
- Functions
- Basic math operations
- Math.round()

## Example

Input array:

[4.2, 5.1, 3.8, 4.7, 5.3]

Output:

Mean: 4.62  
Rounded mean: 5
